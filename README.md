# NewCool Trabajo - Derechos Laborales Chile

Portal educativo sobre trabajo y derechos laborales en Chile.

## Demo

[https://trabajo.newcool.io](https://trabajo.newcool.io)

## Caracteristicas

- Buscador de Inspecciones del Trabajo
- Guia de contratos laborales
- Calculadora de sueldo liquido
- Derechos laborales
- Denuncias laborales

## Tecnologias

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript

## Desarrollo

```bash
npm install
npm run dev
```

## Parte del Ecosistema NewCool

Este modulo es parte de [NewCooltura Informada](https://informada.newcool.io) - Portal de instituciones publicas de Chile.
